На вход программе подаются называния файла с целыми числами, каждое из которых отделено пробелом, и файл, 
в который выводится среднее арифметическое, причем значение в постоянных участках учитывается только один раз.
Имена файлов вводятся с клавиатуры - сначала имя файла со входящими значениями, потом с выходящим.
Если файл с входящими значениям невозможно открыть - программа выдаст ошибку "Failed to open input file".
Если файл с выходящими значениями невозможно открыть - программа выдаст ошибку "Failed to open output file".
Если файл с входящими значенями пуст - программа выдаст ошибку "Input file is empty".
