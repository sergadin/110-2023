На вход программе подается файл с целыми числами, образующими неубывающую последовательность, каждый элемент которой отделен пробелом, 
и файл, в который выводится количество различных элементов последовательности.
Имена файлов вводятся с клавиатуры - сначала имя файла с входящими значениями, потом с выходящим.
Если файл с входящими значениям невозможно открыть - программа выдаст ошибку "Failed to open input file".
Если файл с выходящими значениями невозможно открыть - программа выдаст ошибку "Failed to open output file".
Если файл с входящими значенями пуст - программа выдаст ошибку "Input file is empty".
Если числа образуют не неубывающую последовательность - программа выдаст ошибку "Sequence is not neubivayuschaya".
