На вход программе подается файл с числами, каждое из которых разделено пробелами. Программа выводит в отдельный файл среднее геометрическое значение данных чисел.
Имена файлов вводятся с клавиатуры - сначала файл со входящими значениями, потом с выходящим.
Если файла с входящими значениями нет - программа выдаст ошибку "Failed to open input file".
Если файл с входящими значенями пуст - программа выдаст ошибку "Input file is empty".
Если на вход подается четное кол-во чисел, а их произведение отрицательно - программа выдаст ошибку  "Operation is impossible".
